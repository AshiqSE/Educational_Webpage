Upshot

1. Placements

*Learn Programming: We highly recommend learning C and C++ followed by learning about object orientation and then transitioning to Java. Starting with C will help cover all major programming concepts. It will be very easy to move to any high-level language (like- C++, Java, Python, Java script, etc. ) quite easily if you know C. Though you can directly learn Java or any other highlevel language as well if you want to.

*Basics: Learn different programming concepts like input, output, variables and data types, operators, decision making, loops, arrays, strings and functions.

*Become an expert in Data structures and algorithms: DSA is one of the most foundational topics in computer Science and is the most important topic for interviews. (Ex-Arrays, Strings, Array list, Link list, stack, queue, tree)

*Competitive programming: Website exploring, participate contest

*Development: Websites, Apps, Block chain )

Process to learn code:

1. Take small tutorial.

2. Little bit of experimentation with whatever you learn.

3. Make small project with any instructor.

4. Try to make same projects with some new features from own.

5. If new concepts arise between projects , then take basic idea through documentation detail in initial stage is not required.

6. Revise already learned concepts once in week.

7. Finally, be ready for big exploring.
